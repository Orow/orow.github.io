從 data.json(用陣列包住)中用 ajax 方式獲取資料來渲染成列表。

data.json 格式範例

用 v-for 迴圈帶出背景圖片、標題搭配 a 連結、課程老師。
要用背景圖來渲染，所以用上 v-bind 動態綁定屬性，a 連結的 href 也是動態綁定。

ajax 在 mounted 階段就去初始化，用原生的fetch來串接，之後再用then搭配箭頭函式，把值 return 到原本 vue 實例中 data 的空陣列裡面。
